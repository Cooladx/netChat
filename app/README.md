# net-chat-ui

The user interface of the project.

To install all the dependencies in the app, use

`npm install`

There are other utility npm scripts in the package.json that explain what scripts they will run.

## **Starting the app**

`npm run build` - builds the app with vite which you can then run the files within the build folder that gets generated.

`npm run dev` - runs the app in development mode which you can code along and see the changes as they are happening.
